\## Feature Details



\### Dashboard

Menampilkan ringkasan informasi utama pengguna dan akses cepat ke fitur CyberVault.



\### Cyber Learning Center

Menyediakan materi edukasi keamanan digital seperti phishing awareness, perlindungan data pribadi, dan keamanan akun.



\### Incident Reporting

Memungkinkan pengguna mencatat dan mengirim laporan terkait insiden digital yang dialami.



\### Security Assessment

Membantu pengguna mengetahui tingkat kesiapan keamanan digital melalui asesmen sederhana.



\### Cyber News and Alerts

Menyediakan informasi ancaman siber dan peringatan keamanan digital yang relevan.


## Development Notes

Project ini dikembangkan menggunakan workflow GitHub yang terdiri dari branch, commit, pull request, dan merge agar setiap perubahan terdokumentasi dengan jelas.
